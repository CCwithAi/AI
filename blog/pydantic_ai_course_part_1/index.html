My First Steps with Pydantic AI: Building Simple Agents

Hey everyone! I've been diving into the world of AI agents lately, and I recently stumbled upon something pretty cool called Pydantic AI. If you've used Pydantic for data validation in Python (which I have, and love!), you'll be excited to know that the same team created this new framework specifically for building AI agents. This blog post is all about my initial experiences with it, and how surprisingly easy it is to get started.

My goal here is to share what Iâ€™ve learned, in the hopes it helps others to quickly understand the basics of how Pydantic AI can be used.

What is Pydantic AI and Why Should I Care?

So, what exactly *is* Pydantic AI? In a nutshell, it's a framework designed to make building AI agents simpler and more robust. Think of an AI agent as a little program that can interact with the world, make decisions, and take actions, often using large language models (LLMs) under the hood.

  • Built on a Solid Foundation: It comes from the Pydantic team, and I've found Pydantic to be incredibly useful for ensuring data quality in my Python projects.
  • Model Flexibility: It supports a wide range of LLMs, including OpenAI, Anthropic, Gemini, and even locally-run models like those from Ollama. This means I can choose the best model for my specific needs and budget.
  • Type Safety: Being a bit of a stickler for clean code, I appreciate that Pydantic AI is designed with type checking in mind. This helps catch potential errors early on and makes the code more maintainable.
  • Python-Centric: It leverages Python's strengths, avoiding unnecessary complexity or custom syntax. This makes it feel very natural to work with.
  • Dependency Injection: This might sound a bit technical, but it's a powerful feature that allows you to easily provide data and services to your agents, making testing and development much smoother.

My Pydantic AI Agents

The best way to learn something is by doing, right? So, I jumped right in and started building some basic agents. I was genuinely surprised by how little code it took to get something up and running.

Hello World, AI Agent Style!

My first agent was a classic "Hello World" example. Believe it or not, I got a functioning agent going in just *five* lines of code! Here's how it looked:

    
      from pydantic_ai import Agent
      from pydantic_ai.models.openai import OpenAiModel

      model = OpenAiModel(model_name="gpt-4-0125-preview") # Using GPT-4 for this example
      agent = Agent(model)
      result = agent.run_sync("What is the capital of the United States?")

      print(result.data)

Switching it Up: Using Ollama for Local Models

One of the things I love about Pydantic AI is its flexibility with different models. I wanted to try running a model locally using Ollama, which is a great tool for running open-source LLMs on your own machine.

    
      from pydantic_ai import Agent
      from pydantic_ai.models.ollama import OllamaModel

      model = OllamaModel(model_name="llama3:2b", base_url="http://localhost:11434/v1") # Using Llama 3 2B
      agent = Agent(model)
      result = agent.run_sync("What is the capital of the United States?")

      print(result.data)

Playing with Azure OpenAI

I also have access to Azure OpenAI, and I wanted to see how easy it would be to integrate that. It turns out, Pydantic AI handles this beautifully.

    
      from pydantic_ai import Agent
      from pydantic_ai.models.openai import OpenAiModel
      from openai import AsyncAzureOpenAI
      import os
      from dotenv import load_dotenv

      load_dotenv()

      client = AsyncAzureOpenAI(
          api_key=os.getenv("AZURE_OPENAI_KEY"),
          api_version=os.getenv("AZURE_OPENAI_API_VERSION"),
          azure_endpoint=os.getenv("AZURE_OPENAI_ENDPOINT"),
      )

      model = OpenAiModel(client=client, model_name="gpt-4-turbo") # Using GPT-4 Turbo on Azure
      agent = Agent(model)
      result = agent.run_sync("What is the capital of the United States?")

      print(result.data)

Multi-Agent Magic: Combining Different Models

This is where things got really interesting. I wanted to see if I could create a system where two agents, each using a different model, could communicate with each other. The idea was to have one agent (using OpenAI) answer a question and then pass that answer to a second agent (using Ollama) for further processing.

    
      from pydantic_ai import Agent
      from pydantic_ai.models.openai import OpenAiModel
      from pydantic_ai.models.ollama import OllamaModel
      import os
      from dotenv import load_dotenv

      load_dotenv()

      # OpenAI Agent
      openai_model = OpenAiModel(model_name="gpt-4-0125-preview", api_key=os.getenv("OPENAI_API_KEY"))
      openai_agent = Agent(openai_model)
      openai_result = openai_agent.run_sync("What is the capital of Mexico?")
      print(f"OpenAI Agent: {openai_result.data}")

      # Get the last message from the OpenAI agent
      last_message = openai_result.messages[-1]

      # Ollama Agent
      ollama_model = OllamaModel(model_name="llama3:2b", base_url="http://localhost:11434/v1")
      ollama_agent = Agent(ollama_model)
      ollama_result = ollama_agent.run_sync(
          "Tell me about the history of this city.", message_history=[last_message]
      )
      print(f"Ollama Agent: {ollama_result.data}")

Key Takeaways and Next Steps

My initial exploration of Pydantic AI has been incredibly positive. I'm impressed by its ease of use, flexibility, and the power it offers even for simple agent setups. Here are my main takeaways:

  • Low Barrier to Entry: Getting started is remarkably easy, even for someone relatively new to building AI agents.
  • Model Agnostic: The ability to switch between different LLMs (commercial and open-source) is a huge advantage.
  • Multi-Agent Potential: The ability to create interacting agents opens up exciting possibilities for more sophisticated AI systems.
  • Pythonic Approach: Using a Python-centered approach makes working with Pydantic AI feel very familiar.

Conclusion: Give Pydantic AI a Try!

If you're interested in building AI agents, I highly recommend giving Pydantic AI a try. It's a powerful, flexible, and developer-friendly framework that makes the process surprisingly straightforward. Whether you're a seasoned AI developer or just starting, I think you'll find it a valuable addition to your toolkit.

I'm excited to continue my journey with Pydantic AI and see what I can create. I hope this post has inspired you to explore it as well!
